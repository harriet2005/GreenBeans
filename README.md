GreenBeans is a web-based platform designed to help schools in Kenya organise, track, and showcase environmental activities. Inspired by the principles of Wangari Maathai, the platform focuses on community engagement, youth empowerment, and simple digital tools that make environmental clubs more active and impactful.

What we do: provide students with a simple online platform where they can easily join or connect with existing environmental clubs and communities, participate in verified climate-action activities, and collaborate with partner organisations through weekly or monthly challenges 

Built with React.js for the frontend and Supabase for the backend, GreenBeans gives students and teachers a clean, easy-to-use digital space to manage green clubs, share achievements, participate in challenges, and track real environmental impact.

** Getting Started (For Developers)**

To set up the GreenBeans project locally for development:

### Prerequisites

Ensure you have the following installed:

  * **Node.js** (LTS version recommended)
  * **Git**

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [[YOUR_REPOSITORY_URL_HERE](https://github.com/harriet2005/GreenBeans.git)]
    cd greenbeans
    ```
2.  **Install Frontend Dependencies (React.js):**
    ```bash
    npm install
    ```
3.  **Configure Environment Variables:**
    Create a file named `.env.local` in the root directory and add your Supabase connection details:
    ```
    REACT_APP_SUPABASE_URL=[YOUR_SUPABASE_URL]
    REACT_APP_SUPABASE_ANON_KEY=[YOUR_SUPABASE_ANON_KEY]
    ```
4.  **Run the Development Server:**
    ```bash
    npm start
    ```
    The application will open in your browser, typically at `http://localhost:3000`.
